# Piver Publications — Brand Guidelines

## Logo
- Primary logo: Text-based "PIVER" in uppercase serif font
- Minimum size: 1 inch width
- Clear space: 0.25 inches on all sides
- Logo file provided: piver-logo.svg (see attachments)

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Piver Navy | #1B2A4A | Primary — headings, logo, accents |
| Piver Gold | #C8A951 | Secondary — highlights, call-to-action |
| Warm White | #F5F0E8 | Backgrounds |
| Charcoal | #2D2D2D | Body text |
| Light Gray | #E8E4DC | Borders, dividers |

## Typography

| Use | Font | Weight | Size |
|-----|------|--------|------|
| Headings | Playfair Display | Bold | 24-36pt |
| Subheadings | Playfair Display | Regular | 18-20pt |
| Body | Source Sans Pro | Regular | 10-12pt |
| Captions | Source Sans Pro | Light | 8-9pt |

## Tone
- Professional but approachable
- Confident, not aggressive
- Clear and concise — no jargon
- Focus on value delivered to the reader

## Brand Values
- Quality over quantity
- Clarity in communication
- Trust through transparency
